# Weather App

A React application to check weather information for cities using the OpenWeatherMap API.

## Features

- Search for weather by city name
- Display current temperature, weather description, and icon
- Show additional details like humidity, wind speed, and feels-like temperature
- Loading and error states

## Technologies Used

- React
- Axios
- OpenWeatherMap API

## Setup

1. Get an API key from [OpenWeatherMap](https://openweathermap.org/api)
2. Replace `YOUR_OPENWEATHERMAP_API_KEY` in `src/App.jsx` with your API key
3. Run `npm install`
4. Run `npm run dev` to start the development server

## Installation

1. Clone the repository
2. Run `npm install`
3. Add your OpenWeatherMap API key
4. Run `npm run dev`
